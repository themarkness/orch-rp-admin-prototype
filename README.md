# GOV.UK Prototype

This is a prototype built using the [GOV.UK Prototype Kit](https://github.com/alphagov/govuk-prototype-kit/).

It prototypes possible use cases for RPs

## Prerequisites

- Node.js (version 18 or later recommended)
- npm (comes with Node.js)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/themarkness/rp-admin-tool-protoype.git
   cd codespaces-blank
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the Prototype

To start the development server:

```bash
npm run dev
```

This will start the server on `http://localhost:3000` (or another port if 3000 is in use).

For production serving:

```bash
npm run serve
```
